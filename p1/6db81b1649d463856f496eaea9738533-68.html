Nature of controlling

•

Controlling process is universal

: Controlling is an essential function in any organization such as

an industrial unit, university, government office, hospital etc.,

•

Controlling is a continuous process:

It is a never ending activity on the part of managers and it is

a non stop process. The manager watches the operation of the management and to see whether they
are going towards the desired end and if not, actions are not taken to correct them

•

Controlling is action based

: Action is an essential element of the controlling. Controlling is the

action which ensures the performance according to the decided standards
